Project Overview

Siminds (Virtual Lab) is a cutting-edge educational platform designed to simulate real-world laboratory environments and technical training scenarios. The backend serves as the robust nervous system for this ecosystem, managing the complex lifecycle of virtual experiments, user progressions, and institutional requirements.

It provides immersive learning through virtual kits and 3D simulations, institutional tools for student tracking, and advanced data analytics for insights.

Key Modules

Learning Core

  • Experiments: Interactive learning modules and virtual labs.
  • Courses: Management of educational content and student cohorts.
  • Missions: Gamified learning objectives.

User Management

  • Auth: Isomorphic Auth (JWT + Firebase).
  • Subscriptions: License management and access control.
  • RBAC: Role-based access using CASL.

Engagement

  • Gamification: Leaderboards for competition.
  • Support: Ticketing system and feedback loops.

Data

  • Analytics: User behavior tracking.
  • Reporting: BigQuery integration for heavy reports.

Performance at Scale

High-Performance Caching

Redis reduces primary DB load by 90% and API latency to <10ms for cached content.

Big Data Offloading

Google BigQuery handles terabytes of historical data, executing complex analytical queries in seconds rather than minutes.

Non-Blocking I/O

Node.js event loop handles 10,000+ concurrent connections, ideal for real-time virtual lab states.
